JavaScript Assessment Fetch and Display API Data

Your task is to create a webpage that fetches data from an external API and displays it on the page. Use the JavaScript fetch function to retrieve data from the API endpoint https://jsonplaceholder.typicode.com/posts. Once you have fetched the data, render it in an unordered list (<ul>) on the webpage.
